## React_SWAPI:

### Overview:
Simple web app that fetches Star Wars movies from [SWAPI](https://swapi.dev/) and displays them in a Star Wars inspired theme.

### Deployments:
https://react-swapi-blush.vercel.app/

### Tech Stack:
- **Frontend**: `React.js`, `Tailwind CSS`, `React Query`.
